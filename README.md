1) Download python from the official website and install it
2) pip install -r requirements.txt <- to install dependencies
3) Customize the config for yourself
4) python main.py



Tip:<br> 
If the surnames are moved to 2 lines,<br>
they will move slightly up, just create an additional folder<br>
And run these surnames separately in this folder<br>
You just need to change the shifts in the configs<br>
